# Kryct

> ## ▶ Try it live — **[kryct.com](https://kryct.com)**

[![Live](https://img.shields.io/badge/live-kryct.com-2E72D2)](https://kryct.com)
[![License](https://img.shields.io/badge/license-Proprietary-red)](./LICENSE)

**Kryct is an AI coding platform.** Describe what you want in plain English and
an autonomous agent builds, runs, and ships it — all from the browser. It's a
full cloud IDE: a chat-driven agent, a real editor and shell, a live preview,
and one-click publishing, running on a custom platform built in-house.

## What it does

- **Agentic build loop** — the agent plans, writes files, runs commands, and
  fixes its own errors until the app actually works.
- **Real cloud workspaces** — every project runs in its own isolated, sandboxed
  environment with an editor, shell, and live preview, entirely in the browser.
- **Ship in one click** — publish to a public URL, with optional source backup.
- **Plans & developer API** — metered subscription tiers plus a pay-per-use API
  for programmatic agent runs.

## Under the hood

Kryct runs on a **custom, proprietary platform** designed and operated in-house.
The agent runtime, execution sandbox, usage and limits engine, real-time bridge,
billing, and durable storage are all bespoke, unified behind a single service —
and are not documented publicly.
